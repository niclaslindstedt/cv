# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

This code of conduct applies in all project spaces — GitHub Issues, pull
requests, discussions, and any other venue where contributors interact on
behalf of the project.

## Reporting

To report a violation, use the private contact path described in
[SECURITY.md](SECURITY.md).
